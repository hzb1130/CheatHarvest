# CheatHarvest

## Mod Introduction:
This is a cheat mod that collects nearby items with one click.  
(It was originally designed to pick up items stuck outside the map, which are normally inaccessible, such as cans stuck outside the second-floor door of The Riken at Desolation Point.)
## 模组介绍:
这是一个作弊性质的模组，可一键收集玩家附近的物品。  
(最初设计是用于拾取卡在地图外、正常情况下无法互动的物品，比如荒芜据点的里肯号二层门外面卡着的罐头。）

## Usage Description:
After setting a hotkey, press it to harvest all collectible resources within range — including loose items, plants, and container contents.  
You can toggle collection for each type of resource. Containers will be searched and looted. Already searched containers can be skipped.  
Locked or unopenable containers are ignored.
## 用法描述:
设置热键后，按下即可收集范围内的所有可采集物资，包括散落物品、植物及容器内物资。  
可针对每种资源类型单独开关是否收集。容器将被搜索并取出物资，已搜索过的容器可选择跳过。  
模组会自动屏蔽上锁或无法打开的容器。

## Required Mod:
- ModSettings.dll
## 前置模组需求
- ModSettings.dll

## Warning:
This mod severely disrupts game balance and gameplay.
## 警告：
该模组对游戏平衡和游戏性破坏大。

## Cheat Tip:
Open CheatHarvest.json in the Mods folder and change the pickupRadius value to several thousand for a massive collection range.
## 作弊技巧：
在 Mods 文件夹下打开 CheatHarvest.json，将 pickupRadius 后的数字修改为几千，即可大幅扩展收集范围。
